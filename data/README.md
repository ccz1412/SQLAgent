# dat/ — 数据目录

## 用途
存放项目使用的各类数据集，包括：
- 原始数据文件
- 预处理后的数据
- 数据标注文件
- 数据配置文件

## 注意事项

⚠️ **请勿将大型数据集直接上传至代码仓库！**

数据集、大型数据文件等请通过 **移动存储设备（U盘/移动硬盘）** 交给老师，并在此 README 中记录文件信息：

| 文件名 | 文件大小 | 存放位置 | 备注 |

| （请补充） | | | |

## 数据版本管理建议

如需追踪数据变更，可考虑使用软链接或专门的数据版本管理工具（如 DVC）。
