# dep/ — 依赖目录

## 用途
存放项目所需的第三方依赖工程，包括：
- Git 子模块
- 外部引用库
- 预训练模型文件
- 第三方软件包
- 环境依赖说明

## 注意事项

⚠️ **请勿将大型文件直接上传至代码仓库！**

预训练模型、大型软件包、二进制依赖等请通过 **移动存储设备（U盘/移动硬盘）** 交给老师，并在此 README 中记录文件信息：

| 文件名/包名 | 文件大小 | 存放位置 | 备注 |

| （请补充） | | | |
