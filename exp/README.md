# exp/ — 实验目录

## 用途
存放实验脚本、中间结果和输出数据，可按需包含以下子目录：
- `csv/` — 表格形式的实验结果
- `results/` — 实验结果保存目录
- `eps/` — 矢量图格式结果（也可放于 `vis/`）

## 注意事项

⚠️ **请勿将大型实验结果文件直接上传至代码仓库！**

大型模型输出、训练日志、中间结果等请通过 **移动存储设备（U盘/移动硬盘）** 交给老师，并在此 README 中记录文件信息：
