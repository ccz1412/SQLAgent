## 常见问题排查

### Q1: 本地模型加载失败？
**A**: 确保已安装依赖：
```bash
pip install torch transformers peft accelerate bitsandbytes
```

### Q2: 意图检测不准确？
**A**: 检查 `config/api_config.yaml` 中的 API 配置是否正确。

### Q3: 双模型纠错未生效？
**A**: 检查 `config/agent_config.yaml` 中的 `use_correction_model` 是否设置为 `true`。
